# Data Analytics Portfolio - Alerrandro Heron Ferreira

## Overview

Over the past two years, I have been consistently developing data analysis projects with a strong focus on improving both technical execution and analytical thinking. Until now, this work has remained private.

This repository represents the starting point of a more structured and professional approach to documenting and sharing those projects.

## Purpose

The main objective of this repository is to:

- Centralize previously developed projects
- Apply professional standards in data analysis workflows
- Build a consistent portfolio aligned with real-world business scenarios
- Continuously improve analytical depth and technical implementation

## Approach

All projects included (and future ones) will follow a structured methodology:

1. **Data understanding**
2. **Data cleaning and preparation**
3. **Exploratory analysis**
4. **Metric construction**
5. **Insight generation**
6. **Business-oriented conclusions**

## Tools and Technologies

The work in this repository is based on a core data analytics stack:

- **SQL** — data extraction, transformation, and analysis  
- **Python (pandas)** — data processing and analytical workflows  
- **Power BI** — data visualization and dashboarding  
- **Excel** — quick analysis and validation
